Stellar Level 1 - White Belt dApp
Project Description
This project is a simple Stellar decentralized application (dApp) built on the Stellar Testnet network.
The application allows users to connect their Freighter wallet, view their XLM balance, and send XLM transactions on the Stellar Testnet.
Technologies Used
JavaScript
Vite
Stellar SDK
Freighter Wallet API
Stellar Testnet
Features
Connect Stellar Freighter wallet
Disconnect wallet
Display connected wallet address
Display XLM balance
Send XLM transaction on Stellar Testnet
Show transaction success message and transaction hash
Setup Instructions
To run this project locally:
First, install the required dependencies:
npm install
Then start the development server:
npm run dev
After running the project, connect your Freighter wallet and use the application through the browser.
How It Works
The user first connects their Freighter wallet to the application.
The application retrieves the wallet address and fetches the XLM balance from the Stellar Testnet network.
The user can enter a destination Stellar wallet address and an XLM amount.
When the user starts a transaction, Freighter requests transaction approval.
After approval, the transaction is submitted to the Stellar Testnet network and the transaction hash is displayed as confirmation.
